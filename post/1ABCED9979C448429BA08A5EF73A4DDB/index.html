fcrackzip 破解 带密码的压缩文件

2020-11-27 2 min read # shell

背景 : 想下载一些视频，好不容易找到了，16个G的zip压缩文件，百度云链接。好不容易下载下来了，才发现解压需要密码，然后作者给了一个支付链接...... 看了一下羞涩的钱包，于是尝试用fcrackzip破解

我们可以找到这款应用在 kali tool (kali是一个著名的应用于渗透的linux系统) 网站中。
网站介绍:
fcrackzip is a fast password cracker partly written in assembler. It is able to crack password protected zip files with brute force or dictionary based attacks, optionally testing with unzip its results. It can also crack cpmask’ed images.
This package is useful for pentesters, ethical hackers and forensics experts.
mac 安装：
brew install fcrackzip
查看帮助:
man fcrackzip
通过查看帮助文档 :
发现 -c 参数 可以指定 暴力破解的参数，如数字，大写字母，小写字母，符号。这里只用数字进行破解。如果组合破解，那么如果密码设置过于复杂，暴力破解是出不来的。
先试试水，这玩意管用不管用。

  1. 加密一个压缩文件
    zip -P 12345 a.zip a.txt
  2. 使用fcrackzip来破解
    fcrackzip -b -c '1' -l 1-9 -u a.zip
    结果
    3.使用 fcrackzip 来破解压缩包，发现找不到，于是把 -u 参数去掉,看看可能的密码
    fcrackzip -b -c '1' -l 1-9 target.zip
    然后结果显示
    possible pw found: 139745 ()
    很显然，没有找到正确的密码，用这个密码试一下解压看看。然后竟然解压成功了。 16个G的视频都可以看....wtf;
    解压成功的原因，没弄明白，可能是作者设置的密码就是这个，或者是因为，视频压缩的bug导致解压成功....
